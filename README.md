Retail Sales Data Analysis (Excel Project)

This project focuses on analyzing retail sales data across different stores, product categories, and time periods using Excel.

The dataset includes information such as units sold, revenue, cost, profit, and inventory levels. I started by organizing and cleaning the raw data to make sure everything was consistent and easy to work with. From there, I structured the data so it could be analyzed more efficiently.

I used Excel tools like pivot tables and formulas to break down performance by store (like Tampa and Clearwater), product categories (apparel and footwear), and individual products. This made it easier to see which products were performing well and where profits were coming from.

I also created an analysis sheet and an executive dashboard to summarize the key metrics. The dashboard highlights important insights like revenue trends, profit performance, and sales distribution, making it easier to quickly understand the overall business performance.

Throughout the project, I focused on keeping everything clear and organized so that someone else could open the file and easily follow the analysis without confusion.
